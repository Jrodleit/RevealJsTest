Schere Stein Papier

  • Erstes Spiel, das wir im Fach ITP programmiert haben.
  • Ein einfaches, aber beliebtes Spiel, das auf den Prinzipien von Zufall und Logik basiert.
  • Der Spieler wählt eine der drei Optionen: Schere, Stein oder Papier.

Spielprinzip und Ablauf

Das Spiel funktioniert folgendermaßen:

  • Der Spieler wählt eine der Optionen (Schere, Stein oder Papier).
  • Der Computer wählt zufällig eine der drei Optionen.
  • Der Gewinner wird durch die oben genannten Regeln bestimmt.

Minesweeper

Spielprinzip

Minesweeper ist ein Logikspiel, bei dem der Spieler Felder aufdeckt, ohne auf eine Mine zu klicken. Ziel ist es, alle sicheren Felder zu enthüllen, basierend auf den Zahlen, die die benachbarten Minen anzeigen. Das Spiel endet, wenn entweder alle sicheren Felder freigelegt sind oder eine Mine aufgedeckt wird.

  • Der Spieler klickt auf ein Feld, um es aufzudecken.
  • Jedes Feld zeigt entweder eine Zahl (die Anzahl benachbarter Minen) oder ist leer (kein angrenzendes Minenfeld).
  • Wenn ein Spieler auf eine Mine klickt, endet das Spiel sofort.
  • Das Spiel endet auch, wenn alle sicheren Felder aufgedeckt wurden, ohne eine Mine zu treffen.

Tetris

Spielprinzip

Tetris ist ein klassisches Puzzle-Spiel, bei dem der Spieler fallende Blöcke (Tetrominos) so dreht und positioniert, dass sie eine vollständige Reihe ohne Lücken bilden. Wenn eine Reihe vollständig ist, wird sie gelöscht, und der Spieler erhält Punkte. Das Spiel endet, wenn die Blöcke den oberen Rand des Bildschirms erreichen.

  • Blöcke können nach links, rechts ,unten und gedreht bewegt werden.
  • Es wird ein neuer Block generiert, wenn der der alte block unten ankommt
  • Wenn eine Reihe voll ist, wird sie gelöscht und der Spieler erhält Punkte.

Snake

Spielprinzip

  • Snake ist ein klassisches Arcade-Spiel, bei dem der Spieler eine Schlange steuert, die auf dem Bildschirm wächst, indem sie Äpfel frisst.
  • Die Schlange darf sich nicht selbst beißen oder gegen die Wand stoßen.
  • Das Ziel ist es, so viele Äpfel wie möglich zu essen, um die längste Schlange zu bekommen, ohne das Spiel zu verlieren.

Spielmechanik und Steuerung

  • Die Schlange besteht aus mehreren Segmenten, die dem Kopf folgen.
  • Der Spieler steuert den Kopf der Schlange mit den Pfeiltasten, um die Richtung zu ändern.
  • Wenn die Schlange den Apfel isst, wächst sie um ein Segment.
  • Wenn die Schlange sich selbst oder die Wand berührt, endet das Spiel.
  • Die Schlange bewegt sich in einer der vier Richtungen (oben, unten, links, rechts).
  • Das Spiel überprüft ständig, ob die Schlange auf sich selbst oder auf eine Wand trifft.
  • Für jedes gegessene Apfel wächst die Schlange und die Punktzahl erhöht sich.
